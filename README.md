# 🎬 Telegram Episode Downloader Bot

Railway এ চলমান Telegram Bot যা Episode ডাউনলোড করে।

## 🚀 Features

- ✅ Link দিয়ে Episode ডাউনলোড করুন
- ✅ Playlist তৈরি করুন
- ✅ Download History দেখুন
- ✅ মাল্টিপল ইউজার সাপোর্ট
- ✅ Statistics

## 📋 প্রয়োজন

- Python 3.8+
- Telegram Bot Token
- Railway Account

## 🔧 Local Setup

```bash
git clone https://github.com/আপনারusername/telegram-episode-bot.git
cd telegram-episode-bot
pip install -r requirements.txt
cp .env.example .env
# .env এ BOT_TOKEN যোগ করুন
python main.py
